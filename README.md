# Aplikasi E-Commerce Skincare Emina

Aplikasi e-commerce untuk penjualan produk skincare Emina dengan tema pink.

## Fitur
1. Halaman Login
2. Dashboard
3. Daftar Produk
4. Keranjang Belanja (CRUD)
5. Checkout Pembayaran

## Teknologi
- HTML
- CSS  
- PHP
- JavaScript
- MySQL

## Cara Install
1. Import database `emina_database.sql`
2. Letakkan file di folder htdocs
3. Akses `localhost/namafolder`
